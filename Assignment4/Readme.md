# 📂 Assignment 4 – File Handling

This assignment demonstrates the practical implementation of **Python File Handling** concepts. It covers reading, writing, appending, and processing text files while applying error handling techniques to build reliable file-based applications.

---

## 📌 Objectives

* Read data from text files.
* Write processed data into new files.
* Append information to existing files.
* Process comma-separated values (CSV-style text files).
* Handle file-related exceptions using `try` and `except`.
* Generate reports based on processed data.
